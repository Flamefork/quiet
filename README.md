"Quiet" is a code highlighting theme
- with soft colors and white background
- focused on code, not on coloring

Currently it's for IntelliJ IDEs only.

## Supported languages
- CoffeeScript
- CSS
- Erlang
- Gherkin (Cucumber)
- HAML
- HTML
- JavaScript
- Markdown
- Ruby
- SASS / SCSS
- SQL
- XML
- YAML

## Installation

1. Download `Quiet.xml`
2. Put it in `~/Library/Preferences/<your jetbrains product>/colors` directory.
3. Choose it in `Preferences -> IDE Settings -> Editor -> Colors & Fonts`
